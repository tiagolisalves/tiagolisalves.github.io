Creating a very minimal routing using vanillaJS

When we start to create a single page application we generally rely on a JS framework paired with some routing library. It has a faster setup letting us to focus on the important parts of our applications. But if we were worried about the load time of our app, could we create a routing mechanism using the native api easily? I would say with a loudy voice, yes!
